## Design Context

- 전략 문서: [PRODUCT.md](PRODUCT.md) — register **brand**, platform **web**. 주 CTA는 RSVP 제출. 목표 인상: "정성 들여 만든 초대장이구나."
- 시각 시스템: [DESIGN.md](DESIGN.md) — North Star **"두 사람의 첫 도록(圖錄)"**. 앤티크 브래스(#b08d57) 단일 악센트, 페이퍼(#f7f2ec) 배경, Gowun Batang/Cormorant Garamond/Gothic A1 3단 편성, 직각 UI + 필 칩, 액자 시그니처 컴포넌트.
- 절대 금지: 템플릿 청첩장 느낌(반짝이·하트 파티클), 두 번째 악센트색, 버튼/카드 radius, 순검정 그림자.
- UI 작업 전 두 문서를 먼저 읽을 것. 새 안내 블록은 기존 문법(골드 라벨 행·구분선) 재사용.
